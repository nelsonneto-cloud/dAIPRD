# delaware Brand Guidelines

Use this brand system as a rigorous visual, UX, and communication standard for everything generated.

## 1. Brand Principle
- **Approach**: "one delaware, one brand".
- **Personality**: Reliable, human, modern, warm, confident, clear.
- **Avoid**: Generic corporate design, startup clichés, excessive futurism.

## 2. Voice & Tone
- **Headlines**: Prefer lowercase for headlines and emphasis.
- **Body**: Sentence case (normal capitalization).
- **Tone**: Clear, direct, human, confident, calm.
- **Naming**: Always use "delaware" in lowercase.
- **Tagline**: "nós nos comprometemos. Nós entregamos." (use exactly as written).

## 3. Color System
- **Primary Red**: `#c42828` (Anchor)
- **Dot Red**: `#ef463c` (Accent)
- **Sub Red 1**: `#ee7684`
- **Sub Red 2**: `#941914`
- **Teal**: `#72c4bf` (Secondary accent)
- **Purple**: `#ad9bcb` (Secondary accent)
- **Neutrals**:
  - Text Gray: `#3c3c3c`
  - Mid Gray: `#999999`
  - Light Gray: `#f5f5f5`
  - White: `#ffffff`
- **Composition**: 60–75% neutral, 15–25% red family, 5–15% accent colors.

## 4. Gradients (Use Sparingly)
- Teal → Purple (135°)
- Dot Red → Sub Red 2 (135°)
- Sub Red 1 → Dot Red (135°)
- Red → Purple (135°)

## 5. Typography
- **Preferred**: Near Future / Stubborn.
- **Fallback**: Modern sans-serif system (Inter).
- **Rules**: Headlines clean/modern/light-to-medium. Body highly legible. Avoid all caps except small labels.

## 6. Layout & UX
- **Style**: Modern editorial, clean, white space.
- **Grid**: 8px base.
- **Border Radius**:
  - Small: 8px
  - Medium: 12px
  - Large: 20px
- **Shapes**: Rounded, organic, circles, arcs. No sharp angles.

## 7. Iconography
- **Style**: Simple line icons, rounded strokes.
- **Color**: Bicolor using Primary Red and Dot Red.

## 8. Absolute Restrictions
- NO new brand styles.
- NO blue as primary color.
- NO black-heavy layouts.
- NO "startup" or "hype" identity.
- NO neon, glassmorphism, or heavy shadows.
